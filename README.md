GitHub Developer Analytics Dashboard:

A full-stack web application that analyzes and visualizes developer activity using the GitHub REST API. The application provides insights into GitHub profiles, repository statistics, and programming language distribution through interactive data visualizations.

Live App- 
https://github-dash-board.vercel.app/

Features-

•	Fetch GitHub user profile information
•	Repository statistics (stars, forks, total repositories)
•	Programming language distribution visualization
•	Top repositories leaderboard
•	Loading indicators and error handling
•	Responsive user interface

Tech Stack-

Frontend-
•	React.js
•	Chart.js
•	HTML
•	CSS
•	JavaScript

Backend-
•	Node.js
•	Express.js
•	REST API Integration
